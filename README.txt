## Project Overview

This project demonstrates the deployment of a static website on Amazon Web Services (AWS) using Amazon S3 and Amazon CloudFront. The website files are stored in an Amazon S3 bucket configured for static website hosting and are distributed using Amazon CloudFront for improved performance and availability.
